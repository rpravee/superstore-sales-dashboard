# Superstore Sales Dashboard — Power BI

## Project Overview
An interactive sales dashboard analyzing $2.3M in revenue 
across 4 years using the Kaggle Superstore dataset.

## Key Insights
-  Technology is the top-selling category ($836K)
-  West region leads with 31.58% of total sales
-  Sales grew consistently from 2014 to 2017
-  Copiers have the highest profit margin

## Key Metrics
 Total Sales - $2.30M 
 Profit Margin - 12.47% 
 Total Profit - $286.40K 
 Total Orders - 5,009 
 Avg Order Value - $458.61
